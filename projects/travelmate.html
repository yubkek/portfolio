TravelMate

TravelMate is a conversational trip-planning agent built with LangChain/LangGraph's create agent framework, the LLM autonomously decides which tools to call and in what order, rather than following a hardcoded flow. It integrates Groq-hosted LLMs with real-time flight, hotel, and local-search data via SerpApi, served through a FastAPI backend with a lightweight JS chat frontend. Deployed on Render with session-based conversation state and markdown-formatted, link-rich responses.
